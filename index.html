Projects

  • PAC-MAN Racing

    Basic Javascript experimental gaming project with HTML canvas; gameplay with coin-collect and bot-against

    Tech Stack: JavaScript, jQuery, HTML, Bootstrap

  • Ex.talk

    A full stack web application saves messages to their exes with phone number. If the recipient had also signs up for this app and sends a message to user, the text message will automatically send to user’s phone number. Includes a ‘Meet poeple’ page that shows others stories about their exes and additionally the user can write his/her own story.

    Tech Stack: JavaScript, jQuery, HTML, MongoDB, Node.js, Express, passport, handlebars, Ajax, Bootstrap

  • Cachemon

    This app allows users to scan a QR-code in a physical location and redirect them to the app allowing them to 'catch' a pokemon and add the pokemon to their PokeTeam. Users can also see the location of other QR-codes to find other pokemon.

    Tech Stack: HTML, CSS, JavaScript, Ruby, Ruby on Rails, Bootstrap, jQuery, PokeAPI, Pokegem, Google Maps API, Ajax

  • Doodle 25

    Doodle 25 is a playground where people can join with others and draw anything they like in 60 seconds. Player can go to the Gallery to see others' works or login to save drawings.

    Tech Stack: HTML, CSS, JavaScript, MongoDB, mongoose, Node.js, Express, express-session, passport, handlebars, Socket.io
